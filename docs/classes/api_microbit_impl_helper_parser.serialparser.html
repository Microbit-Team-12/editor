Class SerialParser

This class provides functions to read serial output of a specific procedure, by using the reader helper class.

4|executionStart
5|Code output  |1
6|executionDone
6|MicroPython v1.13 on 2021-02-19; micro:bit v2.0.0-beta.4 with nRF52833
